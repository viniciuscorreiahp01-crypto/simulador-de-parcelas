# Project Context - Simulador de Parcelas

## 1. Product Overview
The **Simulador de Parcelas** is an interactive, web-based tool designed to simplify the calculation of loan installments and interest rates. It provides a clean, modern interface for both lenders and borrowers to visualize payment schedules and total costs in real-time.

## 2. Problem It Solves
Calculating interest-bearing installments manually, especially with specific rounding rules (e.g., to the nearest multiple of R$ 5,00), is prone to error and time-consuming. This application automates these calculations, providing a professional and standardized way to generate loan proposals that can be easily shared via mobile messaging apps.

## 3. Target Users
- **Private Lenders & Small Businesses**: Who need a quick way to show clients the breakdown of their loan payments.
- **Financial Managers**: Handling small-scale credit operations.
- **Borrowers**: Who want to verify the true cost of credit and explore different payment scenarios.

## 4. Core Features
- **Dynamic Simulation**: Instant calculation of installments based on principal and monthly rates.
- **Flexible Ranges**: Ability to simulate multiple installment options (e.g., 1 to 12 months) simultaneously.
- **Schedule Management**: Setting the date for the first payment to generate a precise monthly calendar.
- **Copy-to-Share**: One-click formatting for WhatsApp, improving professional communication.
- **Thematic Consistency**: High-quality dark and light modes with glassmorphism design.

## 5. Technical Context
The application is built using a modern, performant stack:
- **Vite + React 19**: Ensures a fast, responsive user interface.
- **TypeScript**: Provides type safety for numerical calculations.
- **Tailwind CSS 4 + Motion**: Delivers a premium, animated visual experience.

## 6. Future Evolution
The project roadmap includes several high-impact features:
- **Loan Comparison**: Comparing different rates and terms side-by-side.
- **Exporting Options**: Generating PDF or Excel versions of full proposals.
- **Amortization Tables**: Detailed monthly breakdown of principal vs. interest.
- **Financial Scenarios**: Analyzing the impact of early payments or varying interest rates over time.
